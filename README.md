# anura-nvim
Anura theme for vim/neovim

[[image:https://github.com/shMorganson/anura-nvim/blob/main/screenshots/1.png]]

# Prerequisites

Neovim 0.8.0+

# Installing

Using `packer`

```lua
use { "shmorganson/anura-nvim" }
```

Using `lazy.nvim`

```lua
{ "shmorganson/anura-nvim", priority = 1000 }
```
